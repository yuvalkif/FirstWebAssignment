    Q&A



    Question 1: What are the major differences among Web 1.0, 2.0 and 3.0?

    Answer:
     First Difference:

    Interaction between users over the web -

    while Web 1.0 is simply an information portal which can be considerd as the "read only" Web, Web 2.0 and 3.0 support better interaction between users and sites thus it can be considerd as "writable" and allows participation, collaboration, and information sharing unlike Web 1.0.
    Second Difference:

    Interaction between machines and users over the web -

    In Web 3.0, computers can interpret information like humans and intelligently generate and distribute useful content tailored to the needs of user thanks to the semantic web approach which came up after Web 2.0 and 1.0. thus this services not exist in 1.0 and 2.0.
    Third Difference:

    Interaction between machines and machines over the web -

    Web 3.0, with its use of semantics and artificial intelligence is meant to be a “smarter web” and moves toward the Internet of Things. The Internet of Things (IoT) will likely be the next era, Smart devices in the Internet of Things not only use the internet, but speak to each other via machine-to-machine communication (M2M) to accomplish tasks without the need for human input. There are already an abundance of smart devices, as well as automated, internet-connected systems that run without human input. This of course is not supported in Web 1.0 and 2.0



    Question 2: Can you give an example of a difference between Web 1.0 and 3.0?

    Answer: Lets consider this use case of a user who wants to find the side effects of the medicine "x":

    Web 1.0 case:

    1. User inserts the query "x side effects"
    2. Web returns list of static pages
    3. User clicks the first link

    user ends with the full description of medicine x at best or with no results

    Web 3.0 case:

    1. User inserts the query "x side effects"
    2. Web performs query semantic analysis
    3. Web ranks the pages cleverly by relevency
    4. Web returns list of dynamic pages
    5. User clicks the first link

    user ends with a short, proffesional and updated answer by a doctor. solutions to the problem are jumping over the page (by ads, links etc).
